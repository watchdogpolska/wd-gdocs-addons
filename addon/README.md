# addon

[![clasp](https://img.shields.io/badge/built%20with-clasp-4285f4.svg)](https://github.com/google/clasp)

## Setup

```bash
npm install
npm run clasp:login
npm run deploy
```

## Examples

### FederNameByRegon

Values:

```excel
=FederNameByRegon('000678239');
```

Output:

```excel
Bałtycka Galeria Sztuki Współczesnej w Słupsku
```
